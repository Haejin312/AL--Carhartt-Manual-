COLOR DEVELOPMENT & LAB DIP

Lab Dip 제출 (Natific ASAP 사용)
1st: 13일 / Reject 후 재제출: 8일
Vendor → Color Team 피드백: 48시간 내
⌄
✅ 할 일
▸Lab dip 요청 즉시 color standard 구매 → color standard provider에 주문
▸내부 자체 검토 후 최적 submit 1개만 제출 (절대 여러 개 보내지 않음)
▸Submit form 완전히 작성 — Dye Class 필수 기재 (미기재 시 개발 중단)
▸Natific ASAP에 measurable development 업로드
▸Non-measurable: ASAP 시스템 사용하되 'visual' 태그
▸랩딥 사이즈: 최소 5" × 5" (불가 시 2"×4" → 2세트 제출, 사전 승인 필요)
▸2세트 제출 원칙
📐 색상 기준 (Color Criteria)
▸광원: D65 (Primary) / CWF (Secondary) / IncA (Tertiary)
▸Lab Dip 기준: ≤ 0.6 DE CMC (D65, CWF), ≤ 1.5 DE CMC (IncA)
▸Metamerism: ≤ 0.8 Metameric Index
▸Bulk 기준: ≤ 1.0 DE CMC to lab dip (D65, CWF), FR bulk: ≤ 1.5 DE CMC
▸측정기: Datacolor SF600 또는 X-Rite i7 권장
⚠️ 주의사항
⚠️ Carhartt Brown — Best Can Do 승인 절대 없음. 반드시 스펙트럴 + 비주얼 요건 충족
⚠️ Supplier가 자체 QTX 생성 금지 — Carhartt 또는 color standard provider로부터 수령
⚠️ 색상 매칭 위치에는 반드시 full-size color standard 보유 (예외 없음)
💡 SMS 원단 염색은 lab dip 승인 대기 없이 진행 — 마지막 rejected dip 컬러로 진행
